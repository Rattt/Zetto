## Information on release

The version isn't stable yet, to use doesn't sitoit while is in the status of development

## Ruby version tested

2.2.4

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'zetto'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zetto
